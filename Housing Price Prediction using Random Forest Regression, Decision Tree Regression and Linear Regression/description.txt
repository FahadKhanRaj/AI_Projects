  A dataset with 17 distinct rows displaying a specific phone's characteristics was utilized for this research. 16 distinct input features—square meters, number of rooms, numPrevOwners, newly constructed, etc.—and 1 output feature—price range—are included. This issue is a regression oncern as the pricing range is continuous in nature. There are a total of 11 different continuous features and 5 different category characteristics among these 16 input features. The models used in this project are Random Forest Regression, Decision Tree Regression and Linear Regression
